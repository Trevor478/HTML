<blockquote> Rendered as indented text, used for quoting external sources.
<body> Defines the body of an HTML document where the main content is.
<br> Single line brack on the webpage to seperate content.
<button> A button that can be clicked on a website, used with forms.
<canvas> Allows to render 2D images on webpage.
<caption> Placed just after table , used for table caption.
<cite> Used for citing information.
<code> Used for indication a piece of code.

<form> Used for user input.

<h3> Level 3 heading.
<h4> Level 4 heading.
<h5> Level 5 heading.
<h6> Level 6 heading.
<head> The head section. Used mostly for metadata.
<header> Contains introductory content and layout that goes above the body.
<hr> Renders as a horizontal line running across the page, separating text within a paragraph.
<html> The root level tag of an HTML document. All other HTML tags go inside the HTML tag.
<i> Used to make text italicized.

<input> Used for putting forms on your web page.

<li> Represents a list item with an ol (ordered list) or ul (unordered list).

<p> Forms paragraphs within a webpage.
<param> Use when parameters need to be passed to an object embedded within the document.
<pre> For preformatted text. This can be useful for demonstrating code.
<progress> Display the progress of an action, such as with a script.
