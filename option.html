disabled Specifies that an option should be disabled. 
			<html>
			<head>
			<title> Title </title>
			</head>
			<body>

			<option value="mercedes"> Mercedes </option>
			<option value="audi"> Audi </option>
			</select>
			</body>
			</html>
Attributes Description
