<audio>
Attributes Description
autoplay Specifies that the audio will start playing as soon as it is ready. 
			<!DOCTYPE html>
			<html>
			<head>
			<title> Title </title>
			</head>
			<body>
			<audio controls autoplay>

			<source src="horse.ogg" type="audio/ogg">

			<source src="horse.mp3" type="audio/mpeg">

			</audio>
			</body>
			</html>
Attributes Description
controls Specifies that audio controls should be displayed (such as a play/pause button etc). 
			<!DOCTYPE html>
			<html>
			<head>
			<title> Title </title>
			</head>
			<body>
			<audio controls>

			<source src="horse.ogg" type="audio/ogg">

			<source src="horse.mp3" type="audio/mpeg">

			</audio>
			</body>
			</html>
Attributes Description
loop Specifies that the audio will start over again, every time it is finished. 
			<!DOCTYPE html>
			<html>
			<head>
			<title> Title </title>
			</head>
			<body>
			<audio controls loop>

			<source src="horse.ogg" type="audio/ogg">

			<source src="horse.mp3" type="audio/mpeg">

			</audio>
			</body>
			</html>
Attributes Description
muted Specifies that the audio output should be muted. 
			<!DOCTYPE html>
			<html>
			<head>
			<title> Title </title>
			</head>
			<body>
			<audio controls muted>

			<source src="horse.ogg" type="audio/ogg">

			<source src="horse.mp3" type="audio/mpeg">

			</audio>
			</body>
			</html>
Attributes Description
preload Specifies if and how the author thinks the audio should be loaded when the page loads. 
			<!DOCTYPE html>
			<html>
			<head>
			<title> Title </title>
			</head>
			<body>
			<audio controls preload="none">

			<source src="horse.ogg" type="audio/ogg">

			<source src="horse.mp3" type="audio/mpeg">

			</audio>
			</body>
			</html>
Attributes Description
src Specifies the URL of the audio file. 
			<!DOCTYPE html>
			<html>
			<head>
			<title> Title </title>
			</head>
			<body>
			<audio src="horse.ogg" controls>
			</body>
			</html>
