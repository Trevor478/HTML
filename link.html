<link>
Attributes Description
href Specifies the location of the linked document. 
			<html>
			<head>
			<link rel="stylesheet" type="text/css" href="styles.css">
			</head>
			<body>
			<p> Hello World </p>
			</body>
			</html>
Attributes Description
hreflang Specifies the language of the text in the linked document. 
			<html>
			<head>
			<link href="tag_link.asp" rel="parent" rev="subsection" hreflang="en">
			</head>
			<body>
			<p> Hello World </p>
			</body>
			</html>
Attributes Description
media Specifies on what device the linked document will be displayed. 
			<html>
			<head>
			<link rel="stylesheet" type="text/css" href="demo_screen.css">
			<link rel="stylesheet" type="text/css" href="demo_print.css" media="print">
			</head>
			<body>
			<p> Hello World </p>
			</body>
			</html>
Attributes Description
rel Specifies the relationship between the current document and the linked document. 
			<html>
			<head>
			<link rel="stylesheet" type="text/css" href="styles.css">
			</head>
			<body>
			<p> Hello World </p>
			</body>
			</html>
Attributes Description
sizes Specifies the size of the linked resource. Only for rel="icon". 
			<html>
			<head>
			<link rel="icon" href="demo_icon.gif" type="image/gif" sizes="16x16">
			</head>
			<body>
			<p> Hello World </p>
			</body>
			</html>
Attributes Description
type Specifies the media type of the linked document. 
			<html>
			<head>
			<link rel="stylesheet" type="text/css" href="styles.css">
			</head>
			<body>
			<p> Hello World </p>
			</body>
			</html>
