<textarea>
Attributes Description
autofocus Specifies that a text area should automatically get focus when the page loads. 
			<html>
			<head>
			<title> Title </title>
			</head>
			<body>
			<textarea rows="4" cols="50" autofocus>
			This is some text.
			</textarea>
			<p><b> Note: </b> The autofocus attribute of the textarea tag
			is not supported in IE9 and earlier versions. </p>
			</body>
			</html>
Attributes Description
cols Specifies the visible width of a text area. 
			<html>
			<head>
			<title> Title </title>
			</head>
			<body>
			<textarea rows="4" cols="50">
			This is some text.
			</textarea>
			<p> This textarea has 4 visible rows. </p>
			</body>
			</html>
Attributes Description
dirname Specifies that the text direction of the textarea will be submitted. 
			<html>
			<head>
			<title> Title </title>
			</head>
			<body>
			<form action="/action_page.php">
			Text: <br>
			<textarea name:="explanation" 
			dirname="explanation.dir"></textarea>
			<input type="submit" value="Submit">
			</form>
			</body>
			</html>
Attributes Description
disabled Specifies that a text area should be disabled. 
			<html>
			<head>
