<button>
Attributes Description
autofocus Specifies that a button should automatically get focus when the page loads. 
			<!DOCTYPE html>
			<html>
			<head>
			<title> Title </title>
			</head>
			<body>
			<form>			
			<button type="button" autofocus onclick="alert('Hello')"> Click Me </button>
			</form>			
			</body>
			</html>
Attributes Description
disabled Specifies that a button should be disabled. 
			<!DOCTYPE html>
			<html>
			<head>
			<title> Title </title>
			</head>
			<body>
			<form>			
			<button type="button" disabled> Click Me </button>
			</form>			
			</body>
			</html>
Attributes Description
form Specifies one or more forms the button belongs to. 
			<!DOCTYPE html>
			<html>
			<head>
			<title> Title </title>
			</head>
			<body>
			<form>			
			<button type="submit" form="nameform" value="Submit"> Submit </button>
			</form>			
			</body>
			</html>
Attributes Description
formaction Specifies where to send the form-data when a form is submitted. Only for type="submit" 
			<!DOCTYPE html>
			<html>
			<head>
			<title> Title </title>
			</head>
			<body>
			<form>			
			<button type="submit"> Submit </button>
			<button type="submit" formaction="/action_page.php"> Submit to another page.
			</button>
			</form>			
			</body>
			</html>
Attributes Description
formenctype Specifies how form-data should be encoded before sending it to a server. Only for type="submit" 
			<!DOCTYPE html>
			<html>
			<head>
			<title> Title </title>
			</head>
			<body>
			<form>			
			<button type="submit"> Submit with character encoding. </button>
			<button type="submit" formenctype="text/plain"> Submit without character encoding. 
			</button>
			</form>			
			</body>
			</html>
Attributes Description
formmethod Specifies how to send the form-data (which HTTP GET or POST method to use). Only for type="submit" 
			<!DOCTYPE html>
			<html>
			<head>
			<title> Title </title>
			</head>
			<body>
			<form>			
			<button type="submit"> Submit </button>
			<button type="submit" formmethod="post"> Submit using POST. 
			</button>
			</form>			
			</body>
			</html>
Attributes Description
formnovalidate Specifies that the form-data should not be validated on submission. Only for type="submit" 
			<!DOCTYPE html>
			<html>
			<head>
			<title> Title </title>
			</head>
			<body>
			<form>			
			<button type="submit"> Submit </button>
			<button type="submit" formnovalidate> Submit without validation. 
			</button>
			</form>			
			</body>
			</html>
Attributes Description
formtarget Specifies where to display the response after submitting the form. Only for type="submit" 
			<!DOCTYPE html>
			<html>
			<head>
			<title> Title </title>
			</head>
			<body>
			<form>			
			<button type="submit"> Submit </button>
			<button type="submit" formtarget="_blank"> Submit to a new window. 
			</button>
			</form>			
			</body>
			</html>
