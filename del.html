<del>
Attributes Description
cite Specifies a URL to a document that explains the reason why the text was deleted. 
			<!DOCTYPE html>
			<html>
			<head>
			<title> Title </title>
			</head>
			<body>
			<p><del cite="del_demo_cite.htm"> This text has been deleted.
			</del></p>			
			</body>
			</html>
Attributes Description
datetime Specifies the date and time of when the text was deleted. 
			<!DOCTYPE html>
			<html>
			<head>
			<title> Title </title>
			</head>
			<body>
			<p><del datetime="2015-11-15T22:55:03Z"> This text has been deleted.
			</del></p>			
			</body>
			</html>
