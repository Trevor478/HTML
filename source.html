<source>
Attributes Description
src Required when source is used in audio and video. Specifies the URL of the media file. 
			<html>
			<head>			
			</head>
			<body>
			<audio controls>
			<source src="horse.ogg" type="audio/ogg">
			<source src="horse.mp3" type="audio/mpeg">
			Your browser does not support the audio element.
			</audio>
			</body>
			</html>
Attributes Description
srcset Required when source is used in picture. Specifies the URL of the image to use in different situations. 
			<html>
			<head>
			</head>
			<body>
			<picture>
			<source media="(min-width: 650px)" srcset="img_pink_flowers.jpg">
			<source media="(min-width: 465px)" srcset="img_white_flowers.jpg">
			<img src="img_orange_flowers.jpg" alt="flowers" style="width:auto;"> 
			</picture>
			</body>
			</html>
