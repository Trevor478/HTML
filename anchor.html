<anchor>
Attributes Description
download Specifies that the target will be downloaded when a user clicks on the hyperlink. 
			<!DOCTYPE html>
			<html>
			<head>
			<title> Title </title>
			</head>
			<body>
			<p><a href="http://www.google.ca" download="logo"> Google </p>
			</body>
			</html>
Attributes Description
href Specifies the URL of the page the link goes to. 
			<!DOCTYPE html>
			<html>
			<head>
			<title> Title </title>
			</head>
			<body>
			<p><a href="http://www.google.ca"> Google </p>
			</body>
			</html>
Attributes Description
hreflang Specifies the language of the linked document. 
			<!DOCTYPE html>
			<html>
			<head>
			<title> Title </title>
			</head>
			<body>
			<p><a href="http://www.google.ca" hreflang="en"> Google </p>
			</body>
			</html>
Attributes Description
media Specifies what media/device the linked document is optimized for. 
			<!DOCTYPE html>
			<html>
			<head>
			<title> Title </title>
			</head>
			<body>
			<p><a href="http://www.google.ca" media="print and (resolution:300dpi)"> Google </p>
			</body>
			</html>
Attributes Description
ping Specifies a space-separated list of URLs to which, when the link is followed, post requests with the body ping will be sent by the browser (in the background). Typically used for tracking. 
			<!DOCTYPE html>
			<html>
			<head>
			<title> Title </title>
			</head>
			<body>
			<p><a href="http://www.google.ca" ping="http://www.google.ca"> Google </p>
			</body>
			</html>
Attributes Description
rel Specifies the relationship between the current document and the linked document. 
			<!DOCTYPE html>
			<html>
			<head>
			<title> Title </title>
			</head>
			<body>
			<p><a rel="stylesheet" href="main.css" > Main </p>
			</body>
			</html>
Attributes Description
target Specifies where to open the linked document. 
			<!DOCTYPE html>
			<html>
			<head>
			<title> Title </title>
			</head>
			<body>
			<p><a href="http://www.google.ca" target="_blank"> Google </p>
			</body>
			</html>
Attributes Description
type Specifies the media type of the linked document. 
			<!DOCTYPE html>
			<html>
			<head>
			<title> Title </title>
			</head>
			<body>
			<p><a href="http://www.google.ca" type="text/html"> Google </p>
			</body>
			</html>
