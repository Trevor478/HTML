Attributes Description
accept-charset Specifies the character encodings that are to be used for the form submission. 
			<!DOCTYPE html>
			<html>
			<head>
			<title> Title </title>
			</head>
			<body>
			<form action="/action_page.php" accept-charset=ISO-8859-1">
			First Name: <input type="text" name="fname"><br>
			Last Name: <input type="text" name="lname"><br>
			<input type="submit" value="Submit">
			</form>
			</body>
			</html>
Attributes Description
action Specifies where to send the form-data when a form is submitted. 
			<!DOCTYPE html>
			<html>
			<head>
			<title> Title </title>
			</head>
			<body>
			<form action="/action_page.php">
			First Name: <input type="text" name="fname" value="Mickey"><br>
			Last Name: <input type="text" name="lname" value="Mouse"><br>
			<input type="submit" value="Submit">
			</form>
			</body>
			</html>
Attributes Description
autocomplete Specifies whether a form should have autocomplete on or off. 
			<!DOCTYPE html>
			<html>
			<head>
			<title> Title </title>
			</head>
			<body>
			<form action="/action_page.php" method="get" autocomplete="on">

			Email: <input type="email" name="email"><br>
			<input type="submit">
			</form>
			</body>
			</html>
Attributes Description
enctype Specifies how the form-data should be encoded when submitting it to the server (only for method="post") 
			<!DOCTYPE html>
			<html>
			<head>
			<title> Title </title>
			</head>
			<body>
			<form action="/action_page.php" method="post" enctype="multipart/form-data">
			First Name: <input type="text" name="fname"><br>
			Last Name: <input type="text" name="lname"><br>
			<input type="submit">
			</form>
			</body>
			</html>
Attributes Description
method Specifies the HTTP method to use when sending form-data. 
			<!DOCTYPE html>
			<html>
			<head>
			<title> Title </title>
			</head>
			<body>
			<form action="/action_page.php" method="get" target="_blank">

			Last Name: <input type="text" name="lname"><br>
			<input type="submit" value="Submit">
			</form>
			</body>

Attributes Description
name Specifies the name of a form. 
			<!DOCTYPE html>
			<html>
			<head>
			<title> Title </title>
			</head>
			<body>
			<form name="myForm" action="/action_page.php" method="get">

			Last Name: <input type="text" name="lname"><br>

			</form>
			</body>
			</html>
