<embed>
Attributes Description
height Specifies the height of the embedded content. 
			<html>
			<head>
			<title> Title </title>
			</head>
			<body>
			<embed src="helloworld.swf" width="200" height="200" 
			style="border:1px solid">
			<p> Hello World. </p>
			</body>
			</html>
Attributes Description
src Specifies the address of the external file to embed. 
			<html>
			<head>
			<title> Title </title>
			</head>
			<body>
			<embed src="helloworld.swf">			
			<p> Hello World. </p>
			</body>
			</html>
Attributes Description
type Specifies the media type of the embedded content. 
			<html>
			<head>
			<title> Title </title>
			</head>
			<body>
			<embed src="helloworld.swf" type="application/x-shockwave-flash">
			<p> Hello World. </p>
			</body>
			</html>
Attributes Description
width Specifies the width of the embedded content. 
			<html>
			<head>
			<title> Title </title>
			</head>
			<body>
			<embed src="helloworld.swf" width="200" height="200"
			style="border:1px solid">			
			<p> Hello World. </p>
			</body>
			</html>
